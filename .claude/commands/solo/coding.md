Tự chủ implement các task chưa hoàn thành [ ] trong `docs/IMPLEMENTATION_PLAN.md`.

Yêu cầu:
1. Thực hiện song song (parallel agents) các task độc lập để tăng tốc độ
2. Sau khi hoàn thành mỗi task, đánh dấu [x] ngay trong `docs/IMPLEMENTATION_PLAN.md`
3. Báo cáo ngắn gọn sau mỗi milestone quan trọng
4. Nếu gặp ambiguity hoặc conflict với architecture trong `CLAUDE.md`, dừng lại và hỏi — đừng tự assume

Quy tắc:
- Tuân thủ strict layer rules và module pattern trong `CLAUDE.md`
- Không thêm feature ngoài scope IMPLEMENTATION_PLAN
- Ưu tiên làm đúng hơn làm nhanh
